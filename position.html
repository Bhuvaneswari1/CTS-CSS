position: static

An element with position: static, is not positioned in any special way; its always positioned according to the normal flow of the page.

The div element has position: static

position: relative

An element with position: relative; is positioned relative to its normal position;

div element has position: relative;

position: fixed

An element with position:fixed; is positioned relative to the viewport, which means it always stays in the same place if the page is scrolled;

div element has position: fixed

position: absolute

An element with position: absolute; is positioned relative to the nearest positioned ancester(instead of positioned relative to the viewport, like fixed)

The div element has position: relative;
The div element has position: absolute;
